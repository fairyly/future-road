# 如何变得有思想？

日期：2006年12月8日

如果一个人对事物缺乏见解，我们就可以说：“他不是一个有思想的人。”

什么是思想？一个人又怎样可以变得有思想？

首先，思想就是观点。当你对一件事物有观点时，你就是有思想的。根据我的观察，在别人告诉他以前，大多数人其实都是没有观点的。观点越多且越接近本质，那么你的思想也越是丰富和深刻。

这里要注意两个问题。

**（1）观点不是事实。**

观点是基于事实之上的一种系统性判断和理解框架，事实是观点的基础。你了解和掌握的事实越多，就越容易做出自己的判断。但是，事实本身不是观点。我们可以看到有这种人，他非常博学，但是没有思想。

**（2）观点不一定正确。**

错误的思想也是思想。观点既然是一种判断，当然有可能判断错误。现实中，很难判断某种思想是否正确，大多数思想都只是在一定的范围内是正确的。

你可以问一下自己：是否对大多数事物都有自己的观点，而且可以为这些观点辩护（defend），即说出支持这些看法的理由？

如果回答是肯定的，那么你就是有思想的。

那么，如何变得有思想？

人不是生来就有思想的。思想是通过勤奋的学习和思考得来的。我不否认其中存在天赋，某些人生来就是比其他人更有思想。但是，方法论也很重要，掌握正确的方法，有利于形成自己的观点。

要学会有观点，首先必须知道观点的结构。

世界上的所有观点，如果细分一下，无非有三个组成部分。

> （1）事实：又称事实判断，也就是客观的存在，总是成立的，比如“太阳从东方升起”。
>
> （2）前提：又称价值判断，这些判断无法用事实证明，人们假定这些判断是成立的，不需要证明，比如“红色很好看”。
>
> （3）逻辑：推理的规则，用于从判断得到结论。就好像体育比赛必须遵守规则一样，思想也有自己的逻辑规则。逻辑本质上就是数学，所以马克思会说：“任何学科只有到可以用数学表达的阶段，它才是完善的”。

因此，我们有这样一个简单的公式：

> **事实 ＋ 前提 ＋ 逻辑 ＝ 观点**

我做一个简单的分析示范。很多人说应该提高对高收入者的所得税，这个观点是怎么来的？

> （1）事实：高收入者多征税，可以缩小贫富差距。
>
> （2）前提：缩小贫富差距对社会发展有好处。
>
> （3）逻辑：A 可以导致 B，B 可以导致 C，因此 A 可以导致 C。

有了上面的公式，我们就可以明白，所谓没有思想，无非是下面的情况之一。

- 缺乏足够的事实。
- 没有明确的个人价值观（，因此无法形成判断的前提）。
- 逻辑分析能力不够。

反过来也一样，如何变得有思想，无非也是在这几个方面加强努力。

- 了解更多的背景知识。
- 明确个人的价值观，也就是不要有模棱两可的态度，凡事追问自己到底是赞成还是反对，理由何在。
- 培养逻辑分析能力。
